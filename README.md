# Ballet studio - Backend

## Getting started

1.-Install dependencies

```sh
$ npm install
```

2.- up database

```sh
$ docker-compose up -d
```

3.- Run seed - Postman

http://localhost:3000/api/seed/execute-seed

## Running the app

```sh
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
